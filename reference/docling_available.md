# Is the Docling backend available?

Checks whether the `docling` Python package can be imported in the
active 'reticulate' environment.

## Usage

``` r
docling_available()
```

## Value

A logical scalar.

## See also

[`install_docling()`](https://strategicprojects.github.io/doclingr/reference/install_docling.md)

## Examples

``` r
docling_available()
#> Downloading uv...
#> Done!
#> [1] FALSE
```
